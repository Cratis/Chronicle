# Getting Snapshots

> **Status**: This documentation is in progress and will be updated soon.

Learn how to retrieve historical state snapshots of your read models at specific points in time.

## Overview

Chronicle allows you to retrieve snapshots of read models as they existed at any point in the event log. This is powerful for:

- Historical queries and reporting
- Auditing and compliance
- Understanding how state evolved over time
- Time-based analysis
