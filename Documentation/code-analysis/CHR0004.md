# CHR0004: Reactor method has a parameter that cannot be resolved

## Rule Description

A reactor handler method takes the event it reacts to as its first parameter. Any further parameters are resolved as dependencies when the method is invoked:

- The `EventContext` for the triggering event.
- A read model — a type that has a reducer or projection. It is materialized strongly consistent from that reducer or projection.
- A service resolved from the service provider.

A parameter that is a primitive, value type, or string can never be a read model and is almost never a service, so it is flagged: at runtime it would fail to resolve and the partition would fail.

## Severity

Warning

## Allowed Signatures

```csharp
// Event only
void MethodName(TEvent @event)
Task MethodName(TEvent @event)

// Event and context
void MethodName(TEvent @event, EventContext context)
Task MethodName(TEvent @event, EventContext context)

// Event with dependencies (read models and/or services), in any order after the event
Task MethodName(TEvent @event, EventContext context, TReadModel readModel, IService service)
Task MethodName(TEvent @event, TReadModel readModel)
```

Where `TEvent` is a type marked with `[EventType]`, and `TReadModel` is a type that has a reducer or projection.

## Example

### Violation

```csharp
public class OrderReactor : IReactor
{
    // CHR0004: 'count' is an int - a primitive cannot be resolved as a dependency
    public Task OrderUpdated(OrderUpdated @event, int count) => Task.CompletedTask;
}
```

### Fix

```csharp
public class OrderReactor : IReactor
{
    // Take the read model that carries the value instead
    public Task OrderUpdated(OrderUpdated @event, Order order) => Task.CompletedTask;
}
```

## Why This Rule Exists

Reactors take dependencies on their handler methods. Read models are resolved directly by Chronicle, and other dependencies come from the service provider. A primitive, value type, or string is neither, so flagging it early prevents a runtime resolution failure that would pause the event source partition.
